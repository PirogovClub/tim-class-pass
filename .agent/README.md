# Agent Instructions and Workflows

This directory contains instructions and workflows for AI agents (like Antigravity).

## Directory Structure

- **`workflows/`**: Contains task-specific workflows (e.g., `developer.md`).
- **`../skills/`**: Contains complex skill definitions and subagent guidelines.

## How to use

When an agent interacts with this repository, it will look into these directories to understand specific project conventions and follow defined workflows.
Workflows in this directory can be invoked by slash commands if configured.
Skills in the top-level `skills/` directory provide deeper instruction sets.
