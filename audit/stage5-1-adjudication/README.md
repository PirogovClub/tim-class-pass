# Stage 5.1 adjudication — audit submission bundle

Prepared per [`5-1-audit.md`](../../docs/requirements/stage%205/5-1-audit.md), [`5-1-after 1st-audit.md`](../../docs/requirements/stage%205/5-1-after%201st-audit.md), and [`5-1-after-audit2.md`](../../docs/requirements/stage%205/5-1-after-audit2.md).

## One-command refresh (recommended)

From the **repository root**:

```powershell
powershell -File audit/stage5-1-adjudication/package_bundle.ps1
```

This script:

1. Copies **`pipeline/adjudication/*`** and **`tests/test_adjudication_*.py`** into this folder (audit snapshot) and strips bytecode under the bundle.
2. Runs **pytest** (32 adjudication tests) + **compileall** on `pipeline/adjudication` → **`AUDIT_VALIDATION_OUTPUT.txt`**.
3. Copies **only** paths listed in **`$ArchiveRelativePaths`** (in the script) into a **temp staging folder** — same list as [`AUDIT_FILE_LIST.md`](AUDIT_FILE_LIST.md) — strips bytecode again, then zips **staging only**.
4. **Asserts** the zip’s entries exactly match the manifest (fails if anything is missing, extra, or bytecode).
5. Writes **`audit/archives/stage5_1_adjudication_bundle.zip`**.

**`audit/archives/stage5_1_adjudication_bundle.zip`** — rebuild with `package_bundle.ps1`.
