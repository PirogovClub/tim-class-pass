def detect_summary_language(text: str | None) -> str | None:

### Recommended behavior

Return:

* `"ru"` if the text contains Cyrillic and is mostly Russian-script
* `"en"` if the text is Latin-script and looks English
* `None` if empty or ambiguous

Do not use an LLM.
Do not use heavy language detection libraries unless already present.

A simple regex-based heuristic is enough.

Example:

```python
import re

CYRILLIC_RE = re.compile(r"[А-Яа-яЁё]")
LATIN_RE = re.compile(r"[A-Za-z]")

def detect_summary_language(text: str | None) -> str | None:
    if not text:
        return None
    cyr = len(CYRILLIC_RE.findall(text))
    lat = len(LATIN_RE.findall(text))
    if cyr > lat and cyr > 0:
        return "ru"
    if lat > cyr and lat > 0:
        return "en"
    return None
```

Keep it simple.
